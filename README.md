# Rename
Ferramenta de renomeção para agilizar tarefas de gerenciamento de arquivos.

Implementar:

Funcionalidade
  Inserir documentos em uma pasta e ter-los renomeados automaticamente.
  Especificação de prefixo-sufixo
  Contagem de páginas - Renomeação de Cópia Integral
  Identificação automática do documento

Interface gráfica
  Arrastar documentos na interface e terlos nomeados com facilidade
  Definição de prefixo e sufixo COM EXEMPLO
  Boolean para ativação de renomeação automática em pasta selecionada
  Caixa de seleção com tipos(nomes) editaveis
